# 出雲駅伝2025 完全ガイド【エントリーメンバー決定】

2025年10月13日（月・祝）、大学駅伝シーズンの開幕を告げる**出雲駅伝**がついに開催されます！

**全22チームのエントリーメンバーが発表**され、今年も熱い戦いが予想されます。箱根駅伝出場経験者の視点から、注目ポイントを徹底解説します。

## 📅 大会概要

| 項目 | 詳細 |
|------|------|
| 大会名 | 第36回出雲全日本大学選抜駅伝競走 |
| 開催日 | 2025年10月13日（月・祝） |
| スタート時間 | 13:05 |
| コース | 出雲大社前〜出雲ドーム前（45.1km） |
| 区間数 | 6区間 |
| 出場校 | 22チーム |

## 🏃 コースの特徴

出雲駅伝は**スピード勝負**の大会として知られています。

### 区間構成

- **1区（8.0km）**: 出雲大社前スタート。流れを作る重要区間
- **2区（5.8km）**: 最短区間。スピードランナーの見せ場
- **3区（8.5km）**: 大会の趨勢を決める重要区間
- **4区（6.2km）**: 中継所付近の応援が盛り上がる
- **5区（6.4km）**: エース級が投入されることも
- **6区（10.2km）**: アンカー勝負。最長区間でドラマが生まれる

### 箱根駅伝との違い

私が実際に両方の大会を経験して感じた違いは：

1. **距離が短い**: 全体で45.1km。箱根の往路だけで107.5km
2. **スピード重視**: 5000m・10000mの記録が重要
3. **気温が高い**: 10月中旬でまだ暑さが残る
4. **風の影響**: 平坦コースゆえ向かい風が厳しい

## 🏆 優勝予想

私の予想では：

**本命**: 青山学院大学（総合力で一歩リード）
**対抗**: 駒澤大学（安定した強さ）
**穴**: 國學院大學（前回3位の雪辱）
**ダークホース**: 中央大学（吉居・藤田の二枚看板）

ただし、出雲駅伝は短距離ゆえに**1つのミスが命取り**になります。

### 勝負のポイント

1. **1区の流れ**: 出遅れると挽回が難しい
2. **2区のスピード**: 最短区間で差をつける
3. **5区のエース配置**: ここで勝負が決まることも
4. **6区のアンカー勝負**: 最長区間での逆転劇に期待
