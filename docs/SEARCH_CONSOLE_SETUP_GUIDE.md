### 📤 サイトマップ送信手順

#### Step 1: Google Search Consoleにログイン
1. https://search.google.com/search-console にアクセス
2. ekiden-results.com を選択

## Bing Webmaster Tools - サイトマップ送信

### ✅ 前提条件
- [ ] Bing Webmaster Toolsにサイト登録
- [ ] 所有権の確認 (メタタグ追加)

### 📤 サイトマップ送信手順

#### Step 1: Bing Webmaster Toolsにログイン
1. https://www.bing.com/webmasters にアクセス
2. Microsoftアカウントでログイン

#### Step 2: サイトを追加 (初回のみ)
1. 「サイトを追加」をクリック
2. URLを入力: `https://ekiden-results.com`
3. 認証方法を選択: **メタタグ** (既に実装済み)
   ```html
   <meta name="msvalidate.01" content="73F7D8BA7F66661D40546A332A7A8C22" />
   ```
4. 「確認」ボタンをクリック

#### Step 3: サイトマップを送信
1. 左メニューから「サイトマップ」を選択
2. 「サイトマップを送信」ボタンをクリック
3. サイトマップURLを入力:
   ```
   https://ekiden-results.com/sitemap.xml
   ```
4. 「送信」をクリック

#### Step 4: 送信完了確認
- ステータスが「保留中」→「成功」になることを確認
- 検出されたURL数を確認

#### 📍 確認場所
左メニュー → 「ページ」 → 「インデックス作成」

#### 📊 確認項目

##### A. インデックス登録済みページ
**目標:** 660ページ全てがインデックス登録

**確認方法:**
```
Google Search Console → ページ → インデックス登録済み
```

**健全な状態:**
- インデックス登録済み: 660ページ
- インデックス未登録: 0ページ

##### B. カバレッジレポート
**確認項目:**

| ステータス | 説明 | 対応 |
|-----------|------|------|
| ✅ 有効 | 正常にインデックス | 問題なし |
| ⚠️ 警告あり | インデックス登録されているが問題あり | 確認・修正 |
| ❌ エラー | インデックス未登録 | 優先対応 |
| 除外 | 意図的な除外 | 問題なし |

## トラブルシューティング

### 問題1: サイトマップが読み込めない

**症状:**
```
サイトマップを読み込めませんでした
```

**原因と対処:**

#### A. robots.txtの確認
**確認方法:**
```
https://ekiden-results.com/robots.txt
```

**正しい設定:**
```txt
User-agent: *
Allow: /

Sitemap: https://ekiden-results.com/sitemap.xml
```

#### B. sitemap.xmlにアクセス可能か確認
```bash
curl -I https://ekiden-results.com/sitemap.xml
```

**期待されるレスポンス:**
```
HTTP/2 200 
content-type: application/xml
```

#### C. XMLの構文エラー確認
オンラインバリデーターで確認:
- https://www.xml-sitemaps.com/validate-xml-sitemap.html

### 問題2: インデックスが進まない

**症状:**
- 数週間経ってもインデックス数が増えない
- インデックス数が660に達しない

**対処方法:**

#### A. robots.txtの確認
```bash
# ローカルで確認
cat public/robots.txt
```

#### B. 個別URLのインデックスリクエスト
1. Google Search Console → URL検査
2. インデックス未登録のURLを入力
3. 「インデックス登録をリクエスト」をクリック

**注意:** 1日10-15URL程度まで

#### C. サーバーエラーの確認
```bash
# ステータスコード確認
curl -I https://ekiden-results.com/ekiden/hakone/2025
```

### 問題3: クロールエラーが発生

**症状:**
- 404エラー
- 500エラー
- タイムアウト

**対処方法:**

#### A. エラーログの確認
Google Search Console → カバレッジ → エラー詳細

#### B. 404エラーの対処
```typescript
// app/not-found.tsx で適切な404ページを実装済み
```

#### C. サーバーレスポンスの改善
- ビルド時の静的生成確認
- CDN設定の確認
